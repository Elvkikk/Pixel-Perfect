# Pixel Perfect

A React, TypeScript, and Vite project recreating the npm homepage with a focus on visual accuracy.

## Tech Stack

- React
- TypeScript
- Vite
- Sass modules
- Tailwind CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run linting:

```bash
npm run lint
```
